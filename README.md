# Graph Neural Network based Emotion Recognition in Conversation

This is the open sourced code for the final project of the course CSCI 662 Fall 2020.

## Requirement

The code requires the following packages to run:

* python >= 3.8.5
* numpy >= 1.19.2
* tqdm >= 4.50.2
* pytorch >= 1.7.0
* transformers >= 4.0.0
* scipy >= 1.5.2
